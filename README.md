# Decodelabs
Project Overview :
This project focuses on cleaning, preparing, and analyzing an e-commerce dataset containing customer orders, product details, payment information, shipping details, and sales records. The main objective was to improve data quality and prepare the dataset for accurate analysis and business insights.

Work Done :
- Imported and explored the raw dataset.
- Analyzed customer orders, products, payments, and sales information.
- Identified data quality issues affecting analysis.
- Cleaned and transformed the dataset to improve reliability.
- Verified sales calculations and order information.
- Performed exploratory data analysis to understand trends and patterns.
- Generated insights related to products, customers, and order performance.
- Prepared the final cleaned dataset for future reporting and machine learning applications.
  
Data Cleaning Process :
- Removed duplicate records to maintain data consistency.
- Checked for missing and null values in all columns.
- Corrected inconsistent data formats and values.
- Validated order IDs and customer IDs.
- Verified product quantities and unit prices.
- Recalculated total prices and compared them with recorded values.
- Standardized categorical fields such as payment methods and order status.
- Ensured tracking information and shipping details were properly formatted.

Data Preparation :
- Converted data into a structured and analysis-ready format.
- Processed date fields for time-based analysis.
- Organized customer, product, and transaction information.
- Created clean features suitable for visualization and predictive modeling.
- Ensured data integrity across all records.
- Dataset Features

The dataset includes :
- Order Information
- Customer Information
- Product Details
- Quantity and Pricing
- Payment Methods
- Order Status
- Shipping Information
- Coupon Usage
- Referral Sources
- Sales and Revenue Data

Key Outcome :
The project successfully transformed raw e-commerce transaction data into a clean, structured, and reliable dataset that can be used for business intelligence, sales analysis, customer behavior analysis, and future machine learning applications.
